# zhenxun_plugin_ddcheck

zhenxun_bot 成分姬插件

移植自[nonebot-plugin-ddcheck](https://github.com/noneplugin/nonebot-plugin-ddcheck)

安装相关依赖

```
pip3 install -r requirements.txt
```

### 使用

```
查成分 + B站用户名/UID
```

若要显示主播牌子，需要在 `configs\config.yaml` 文件中添加任意的B站用户cookie：

```
BILIBILI_COOKIE: xxx
```
